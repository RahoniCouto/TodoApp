# TodoApp
Developed based on the React course from Udemy- by Leonardo Moura Leitao.

## How to install packages

### after cloning the repository enter the folder

```sh
$ cd backend
```

### yarn

```sh
$ yarn
$ yarn dev
```

### npm

```sh
$ npm install
$ npm start
```

## How to run the BackEnd application

### yarn

```sh
$ yarn dev
```

### npm

```sh
$ npm start
```
